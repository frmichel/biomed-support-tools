This folder provides tools for monitoring SEs:
- monitor the total and used space of all SEs
    - collect-se-space.py: collect the data from the BDII for each SE
    - run-collect-se-space.sh: wrapper to run collect-se-space.py
    - process-se-space.py: process the data acquired by collect-se-space.py to produce reports
    - processors/: compute different views and distribution
