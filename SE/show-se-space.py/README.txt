This folder provides tools for monitoring SEs:
- monitor thei total and used space of all SEs
    show-se-space.py: collect the data from the BDII for each SE
    run-monitor-se.sh: wrapper to run show-se-space.py
    process-se-space.py: process the data acquired by show-se-space.py to produce reports
    processors/: compute different views and distribution
